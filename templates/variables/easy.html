Problem 1: Personal Information

Create variables to store personal information

Task:

Create variables to store your name, age, city, and favorite programming language. Then print a formatted message introducing yourself.

Solution:

# Store personal information
name = "John Doe"
age = 25
city = "New York"
favorite_language = "Python"

# Print introduction
print(f"Hi! I'm {name}, {age} years old.")
print(f"I live in {city} and I love {favorite_language}!")

# Alternative formatting
introduction = f"My name is {name}. I am {age} years old, live in {city}, and my favorite programming language is {favorite_language}."
print(introduction)

Expected Output:

Hi! I'm John Doe, 25 years old.
I live in New York and I love Python!
My name is John Doe. I am 25 years old, live in New York, and my favorite programming language is Python.

Problem 2: Simple Calculator

Practice basic arithmetic with variables

Task:

Create two number variables and perform all basic arithmetic operations (addition, subtraction, multiplication, division).

Solution:

# Define two numbers
num1 = 15
num2 = 4

# Perform arithmetic operations
addition = num1 + num2
subtraction = num1 - num2
multiplication = num1 * num2
division = num1 / num2
floor_division = num1 // num2
modulus = num1 % num2
power = num1 ** num2

# Display results
print(f"Number 1: {num1}")
print(f"Number 2: {num2}")
print(f"Addition: {num1} + {num2} = {addition}")
print(f"Subtraction: {num1} - {num2} = {subtraction}")
print(f"Multiplication: {num1} * {num2} = {multiplication}")
print(f"Division: {num1} / {num2} = {division}")
print(f"Floor Division: {num1} // {num2} = {floor_division}")
print(f"Modulus: {num1} % {num2} = {modulus}")
print(f"Power: {num1} ** {num2} = {power}")

Expected Output:

Number 1: 15
Number 2: 4
Addition: 15 + 4 = 19
Subtraction: 15 - 4 = 11
Multiplication: 15 * 4 = 60
Division: 15 / 4 = 3.75
Floor Division: 15 // 4 = 3
Modulus: 15 % 4 = 3
Power: 15 ** 4 = 50625

Problem 3: Variable Swapping

Learn different ways to swap variable values

Task:

Create two variables with different values and swap them using different methods.

Solution:

# Method 1: Using a temporary variable
a = 10
b = 20
print(f"Before swap: a = {a}, b = {b}")

temp = a
a = b
b = temp
print(f"After swap (temp): a = {a}, b = {b}")

# Method 2: Python tuple unpacking (most Pythonic)
x = 100
y = 200
print(f"Before swap: x = {x}, y = {y}")

x, y = y, x
print(f"After swap (tuple): x = {x}, y = {y}")

# Method 3: Using arithmetic (only for numbers)
p = 5
q = 15
print(f"Before swap: p = {p}, q = {q}")

p = p + q  # p = 5 + 15 = 20
q = p - q  # q = 20 - 15 = 5
p = p - q  # p = 20 - 5 = 15
print(f"After swap (arithmetic): p = {p}, q = {q}")

Expected Output:

Before swap: a = 10, b = 20
After swap (temp): a = 20, b = 10
Before swap: x = 100, y = 200
After swap (tuple): x = 200, y = 100
Before swap: p = 5, q = 15
After swap (arithmetic): p = 15, q = 5

Problem 4: String Manipulation

Practice working with string variables

Task:

Create string variables for first name and last name. Combine them and display in different formats.

Solution:

# String variables
first_name = "Alice"
last_name = "Johnson"

# Combine names in different ways
full_name = first_name + " " + last_name
formal_name = last_name + ", " + first_name
initials = first_name[0] + "." + last_name[0] + "."

# Display information
print(f"First Name: {first_name}")
print(f"Last Name: {last_name}")
print(f"Full Name: {full_name}")
print(f"Formal Name: {formal_name}")
print(f"Initials: {initials}")

# String operations
print(f"Full name length: {len(full_name)} characters")
print(f"Uppercase: {full_name.upper()}")
print(f"Lowercase: {full_name.lower()}")
print(f"Title case: {full_name.title()}")

# Check if name contains certain letters
contains_a = "a" in full_name.lower()
print(f"Contains letter 'a': {contains_a}")

Expected Output:

First Name: Alice
Last Name: Johnson
Full Name: Alice Johnson
Formal Name: Johnson, Alice
Initials: A.J.
Full name length: 13 characters
Uppercase: ALICE JOHNSON
Lowercase: alice johnson
Title case: Alice Johnson
Contains letter 'a': False
