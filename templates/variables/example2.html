Example 2: Variable Types

Discover different data types in Python

Python Variable Types

Python automatically determines the type of a variable based on the value assigned to it. You can use the type() function to check the data type.

Python Code

# Different variable types
a = 10          # Integer
b = 3.14        # Float
c = True        # Boolean
d = "Python"    # String
e = [1, 2, 3]   # List

# Check the type of each variable
print(f"a = {a}, type: {type(a)}")
print(f"b = {b}, type: {type(b)}")
print(f"c = {c}, type: {type(c)}")
print(f"d = {d}, type: {type(d)}")
print(f"e = {e}, type: {type(e)}")

Expected Output:

a = 10, type: <class 'int'>
b = 3.14, type: <class 'float'>
c = True, type: <class 'bool'>
d = Python, type: <class 'str'>
e = [1, 2, 3], type: <class 'list'>

Integer (int)

Whole numbers without decimal points

Example: 10, -5, 0

Float

Numbers with decimal points

Example: 3.14, -2.5, 0.0

Boolean (bool)

True or False values

Example: True, False

String (str)

Text data in quotes

Example: "Hello", 'Python'

Key Points

  • Dynamic Typing: Python automatically determines variable types
  • Type Checking: Use type() function to check variable types
  • Type Conversion: You can convert between types using int(), float(), str(), etc.
  • Immutable Types: int, float, bool, and str cannot be changed in place
