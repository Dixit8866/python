Variables in Python

Learn how to store and work with data using variables

What are Variables?

Variables are containers that store data values. In Python, you don't need to declare the type of variable explicitly - Python automatically determines the data type based on the value you assign.

Think of variables as labeled boxes where you can store different types of information that your program can use later.

Key Concepts

  • Variables store data that can be used throughout your program
  • Python is dynamically typed - no need to declare variable types
  • Variable names should be descriptive and follow naming conventions

Common Errors & How to Fix Them

NameError: name 'variable' is not defined

Occurs when you try to use a variable before defining it.

Wrong: print(x) # x not defined
Correct: x = 5; print(x)

Invalid variable names

Variable names must follow Python naming rules.

Wrong: 2var, class, my-var
Correct: var2, my_class, my_var
