Example 1: Basic Assignment

Learn how to assign values to variables

Understanding Variable Assignment

In Python, you create variables by simply assigning a value to a name. The assignment operator (=) is used to store data in variables.

Python Code

# Assigning values to variables
x = 5
y = "Hello"
name = "Python"
score = 98.5

# Printing the variables
print(x)        # Output: 5
print(y)        # Output: Hello
print(name)     # Output: Python
print(score)    # Output: 98.5

Expected Output:

5
Hello
Python
98.5

Explanation

  • x = 5: Creates an integer variable named 'x' with value 5
  • y = "Hello": Creates a string variable named 'y' with value "Hello"
  • name = "Python": Creates a string variable for storing names
  • score = 98.5: Creates a float variable for decimal numbers
