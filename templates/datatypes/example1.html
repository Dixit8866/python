Example 1: Numeric Types

Working with integers and floating-point numbers

Understanding Numeric Data Types

Python supports several numeric data types. The most commonly used are integers (int) and floating-point numbers (float). Python automatically handles the precision and memory allocation.

Python Code

# Integer examples
age = 25
population = 1000000
negative_num = -42

# Float examples  
price = 19.99
temperature = -5.5
pi = 3.14159

# Check types and perform operations
print(f"age: {age}, type: {type(age)}")
print(f"price: {price}, type: {type(price)}")

# Mathematical operations
total = age + 5
discount = price * 0.1
area = pi * (5 ** 2)  # π * r²

print(f"Total: {total}")
print(f"Discount: ${discount:.2f}")
print(f"Area: {area:.2f}")

# Type conversion
age_as_float = float(age)
price_as_int = int(price)

print(f"Age as float: {age_as_float}")
print(f"Price as int: {price_as_int}")

Expected Output:

age: 25, type: <class 'int'>
price: 19.99, type: <class 'float'>
Total: 30
Discount: $2.00
Area: 78.54
Age as float: 25.0
Price as int: 19

Key Points

  • Integers: Whole numbers without decimal points (unlimited precision)
  • Floats: Numbers with decimal points (64-bit precision)
  • Type Conversion: Use int() and float() to convert between types
  • Precision: Be careful with float precision in calculations
  • Operations: Mix int and float - result is usually float
