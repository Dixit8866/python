Example 2: String Operations

Working with text data and string manipulation

Understanding Strings

Strings are sequences of characters used to represent text. In Python, strings are immutable, meaning they cannot be changed after creation. However, you can create new strings based on existing ones.

Python Code

# Creating strings
name = "Python"
language = 'Programming'
message = """This is a
multi-line string"""

# String information
print(f"Name: {name}")
print(f"Type: {type(name)}")
print(f"Length: {len(name)}")

# String operations
greeting = "Hello, " + name + "!"
repeated = name * 3
uppercase = name.upper()
lowercase = name.lower()

print(f"Greeting: {greeting}")
print(f"Repeated: {repeated}")
print(f"Uppercase: {uppercase}")
print(f"Lowercase: {lowercase}")

# String methods
sentence = "  Python is awesome!  "
cleaned = sentence.strip()
words = cleaned.split()
replaced = cleaned.replace("awesome", "amazing")

print(f"Original: '{sentence}'")
print(f"Cleaned: '{cleaned}'")
print(f"Words: {words}")
print(f"Replaced: '{replaced}'")

# String formatting
age = 30
formatted = f"I am {age} years old and I love {name}!"
print(formatted)

# Checking string content
print(f"Starts with 'Py': {name.startswith('Py')}")
print(f"Contains 'th': {'th' in name}")
print(f"Is alphabetic: {name.isalpha()}")

Expected Output:

Name: Python
Type: <class 'str'>
Length: 6
Greeting: Hello, Python!
Repeated: PythonPythonPython
Uppercase: PYTHON
Lowercase: python
Original: ' Python is awesome! '
Cleaned: 'Python is awesome!'
Words: ['Python', 'is', 'awesome!']
Replaced: 'Python is amazing!'
I am 30 years old and I love Python!
Starts with 'Py': True
Contains 'th': True
Is alphabetic: True

Common Methods

upper() - Uppercase

lower() - Lowercase

strip() - Remove whitespace

split() - Split into list

Testing Methods

startswith() - Starts with

endswith() - Ends with

isalpha() - All letters

isdigit() - All digits

Key Points

  • Immutable: Strings cannot be changed, only replaced
  • Quotes: Use single ('') or double ("") quotes
  • Concatenation: Use + to join strings or f-strings for formatting
  • Indexing: Access characters with [index], starting from 0
  • Methods: Strings have many built-in methods for manipulation
